This blog now lives as **Markdown/MDX** next to the site source. There is no separate CMS or API for posts: CI builds the site and GitHub Pages serves static HTML.

Comments use **[giscus](https://giscus.app/)**, which stores threads in **GitHub Discussions** and lets readers sign in with a GitHub account.

## Migrating older posts

If you still have articles in the old Mongo-backed API, export each post to a file under `src/content/blog/` with frontmatter fields matching the collection schema (`title`, `pubDate`, `author`, optional `description`, `heroImage`, `tags`). The body can be Markdown or MDX.

## Images

You can keep **Cloudinary** URLs in frontmatter and markdown, or place files in `public/` and reference them with root-relative paths.
